## 🚀 開始練習

```bash
# 複製專案
git clone https://github.com/Jeffrey0117/react-beginner-challenges.git
cd react-beginner-challenges

# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev
```

## 💡 如何使用

1. 進入各題目資料夾查看 README
2. 先看 `problem/` 資料夾的問題程式碼
3. 嘗試自己解決問題
4. 查看 `solution/` 資料夾的解答
