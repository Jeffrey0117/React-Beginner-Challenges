## 🚀 開始練習

```bash
# 複製專案
git clone https://github.com/Jeffrey0117/react-beginner-challenges.git
cd react-beginner-challenges

# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev
```

## 💡 如何使用

1. **選擇章節**：從 CHAPTER 01 開始，循序漸進
2. **進入題目**：點擊題目連結查看詳細說明
3. **查看問題**：先看 `problem/` 資料夾的問題程式碼
4. **嘗試解決**：自己動手實作解決方案
5. **查看解答**：參考 `solution/` 資料夾的解答
6. **理解原理**：搞懂為什麼這樣寫才是對的
